# kenherring/openedge-actions-samples

This repository contains sample GitHub Actions workflows that use [kenherring/openedge-actions](https://github.com/kenherring/openedge-actions).

## Sample Code

| Workflow | Description |
| -------- | ----------- |
| [![CI](https://github.com/kenherring/openedge-actions-samples/actions/workflows/run_a_program.yml/badge.svg)](.github/workflows/run_a_program.yml) | Run OpenEdge programs using the `run` action |
| [![CI](https://github.com/kenherring/openedge-actions-samples/actions/workflows/run_ablunit_tests.yml/badge.svg)](.github/workflows/run_ablunit_tests.yml) | Run ABLUnit tests using the `ablunit` action |
| [![CI](https://github.com/kenherring/openedge-actions-samples/actions/workflows/create_sports_database.yml/badge.svg)](.github/workflows/create_sports_database.yml) | Setup openedge, configure a database, and then run code/tests that connect to that database |
