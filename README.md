# kenherring/openedge-actions-samples

This repository contains sample GitHub Actions workflows that use [kenherring/openedge-actions](https://github.com/kenherring/openedge-actions).

## Sample Code

| Workflow | Description |
| -------- | ----------- |
| [run_a_program](.github/workflows/run_a_program.yml) | Run OpenEdge programs using the `run` action |
| [run_ablunit_tests](.github/workflows/run_ablunit_tests.yml) | Run ABLUnit tests using the `ablunit` action |
| [create_sports_database](.github/workflows/create_sports_database.yml) | Setup openedge, configure a database, and then run code/tests that connect to that database |
